# 健康趋势分析器 - 使用示例

## 示例 1：一般健康趋势分析

### 用户输入
```
过去3个月我的健康有什么变化？
```

### Claude 执行

1. **确定时间范围**：过去3个月（2025-10-01 至 2025-12-31）
2. **读取数据**：
   - profile.json → 体重历史：70kg → 68.5kg
   - symptoms/ → 头痛12次、疲劳8次、失眠6次
   - mood/ → 平均情绪评分6.8/10，平均睡眠6.5小时
   - medication-logs/ → 依从率78%（漏服8次）
   - medical_records/ → 胆固醇240→210，血糖5.6→5.4
3. **趋势分析**：
   - 体重：下降2.3kg（-3.2%），积极趋势
   - 症状：头痛频率下降（较上期减少4次）
   - 化验：胆固醇改善（-30mg/dL）
   - 依从性：氨氯地平仅65%，需改进
4. **相关性分析**：
   - 睡眠时长 ↔ 情绪评分：r=0.78（强正相关）
   - 用药依从性 ↔ 症状频率：r=-0.62（中等负相关）
5. **生成HTML报告**：保存至 `data/health-reports/health-trend-report-2025-12-31.html`

### 输出结果

**文本报告**：

   - 保持水分充足

4. **监测**：
   - 继续记录头痛日记
   - 1个月后评估改善情况
   - 如无改善或加重，建议就医

2. **读取血压数据**：
   - 用药前（9月）：平均158/95 mmHg
   - 用药后1个月（10月）：平均142/88 mmHg
   - 用药后2个月（11月）：平均135/85 mmHg
   - 用药后3个月（12月）：平均132/82 mmHg

3. **读取症状数据**：
   - 头晕（用药相关症状）：用药前8次/月 → 用药后3次/月
   - 心悸：用药前5次/月 → 用药后1次/月

4. **效果评估**：
   - **血压控制**：收缩压下降26mmHg，舒张压下降13mmHg ✅
   - **达标率**：从0%提升至75%（9次/12次测量<140/90）
   - **症状改善**：头晕、心悸频率明显下降 ✅

5. **依从性分析**：
   - 总体依从率：65%⚠️（目标≥90%）
   - 漏服模式：晚间剂量漏服较多（20次 vs 早晨7次）

1. **总体偏低**：65%依从率低于目标90%
2. **晚间漏服**：晚间剂量漏服率高达52%（20次/39次）

**1. 提高晚间依从性**：
- 设置晚间用药提醒（建议20:00，晚饭后30分钟）
- 将药物放在显眼位置（餐桌、床头柜）
- 使用分药盒提前准备一周药物

- 如发现漏服距原计划时间<2小时，可立即补服
- 如已超过2小时，跳过本次剂量，按原计划继续
- 避免下次服用双倍剂量

**3. 监测计划**：
- 继续每日监测血压，记录日志
- 1个月后评估血压控制稳定性和依从性改善
- 如依从性提升至90%但血压仍未达标，建议就医调整剂量

**4. 生活方式配合**：
- 低盐饮食（<6g/天）
- 规律运动（每周150分钟中等强度）
- 限制饮酒（男性<25g/天，女性<15g/天）

## 示例 5：相关性分析

### 用户输入
```
我的情绪和睡眠有什么关系？分析相关性
```

### Claude 执行

1. **读取情绪数据**（过去3个月，90天）：
   - 有效记录：82天（91%记录率）
   - 情绪评分范围：3-10分
   - 平均情绪评分：6.8/10
   - 情绪波动：标准差1.2

2. **读取睡眠数据**：
   - 睡眠时长范围：4.5-9小时
   - 平均睡眠时长：6.5小时
   - 睡眠质量：良好（6次）、中等（48次）、差（28次）

3. **相关性计算**：
   - **睡眠时长 ↔ 情绪评分**：r = 0.78（强正相关）
   - **睡眠质量 ↔ 情绪评分**：r = 0.65（中等正相关）
   - **压力评分 ↔ 情绪评分**：r = -0.52（中等负相关）
   - **运动时间 ↔ 情绪评分**：r = 0.43（弱正相关）

4. **模式识别**：
   - **高情绪天数**（≥8分）：平均睡眠7.2小时
   - **低情绪天数**（≤5分）：平均睡眠5.4小时

### 分析与建议

**🔗 强相关性发现**：
1. **睡眠时长与情绪**：相关系数0.78（强正相关）

2. **睡眠质量与情绪**：相关系数0.65
   - 睡眠质量"良好" → 平均情绪7.9分
   - 睡眠质量"差" → 平均情绪5.3分

1. **低谷期特征**（10月中旬）：
   - 连续5天情绪≤5分
   - 平均睡眠仅5.2小时
   - 工作压力评分高（8/10）
   - 缺乏运动和社交活动

2. **高峰期特征**（12月上旬）：
   - 连续7天情绪≥8分
   - 平均睡眠7.5小时
   - 工作压力低（3/10）
   - 运动和社交活动充足

**💡 改善建议**：

**1. 优先级：提高睡眠时长**（影响最大）
- **目标**：从6.5小时提升至7-8小时
- **方法**：
  - 固定上床时间：22:30-23:00
  - 固定起床时间：6:30-7:00
  - 睡前1小时避免屏幕（蓝光影响褪黑素）
  - 创造良好睡眠环境（黑暗、安静、凉爽）

**2. 建立规律作息**
- 周末不熬夜补觉（保持生物钟稳定）
- 午休限制在20-30分钟（避免影响夜间睡眠）
- 建立睡前仪式（阅读、冥想、温水澡）

**3. 压力管理**
- 学习深呼吸、冥想技巧
- 每周安排放松活动（瑜伽、太极、散步）
- 培养兴趣爱好（音乐、绘画、园艺）
- 保持社交联系（朋友、家人）

**4. 增加运动**（辅助改善）
- 每周至少150分钟中等强度运动
- 运动日情绪平均高0.7分
- 避免睡前3小时剧烈运动

**5. 监测与调整**
- 继续记录情绪和睡眠（保持高记录率91%）
- 2周后评估改善情况
- 如无改善，考虑专业心理咨询

**📈 预期改善**：
- 睡眠提升至7-8小时 → 情绪评分预计提高至7.5-8.0分
- 建立规律作息 → 情绪波动减少（标准差降至0.8以下）
