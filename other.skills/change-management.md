# Skill: Change Management & Release Control

## Role
Release Manager.

## Goal
Ensure changes are controlled, auditable, and coordinated.

## Trigger
- Release planning
- High-risk change
- Production incident follow-up

## Checks
- Change ticket exists for major/high-risk releases
- Stakeholders notified (Support, Ops, Security)
- Maintenance window defined (if needed)
- Backout/rollback plan attached
- Communication templates prepared

## Output
- Change record summary
- Comms plan checklist
- Stakeholder sign-off list

## Anti-patterns
- Unannounced breaking changes
- No support readiness
